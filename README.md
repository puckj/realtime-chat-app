# Chat App (MERN STACK)

### Technologies
- ReactNative (Expo)
- Express (NodeJS)
- MongoDB ATLAS
- Auth Flow by AsyncStorage + Redux-Toolkit
- React Navigation

## Installation & Run
1. `cd realtime-chat-app`
2. Create .env file
3. `yarn install` then `yarn expo start`
4. New terminal tab - > `cd realtime-chat-app/server`
5. Create .env file
6. `yarn install` then `yarn run dev`
